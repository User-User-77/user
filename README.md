# BRTS QR Ticket

A React-based mobile QR ticket application for BRTS bus service.

## Features

- QR Code display for ticket validation
- Live countdown timer (7 PM - 10 PM)
- Collapsible ticket details
- Mobile-responsive design (375px width)
- Modern UI with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- qrcode.react

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for Netlify deployment.
